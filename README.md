# Contador de Calorías

Aplicación web para el registro y control de calorías consumidas mediante alimentos y calorías quemadas a través de ejercicios. El objetivo es mostrar un balance claro que ayude a mejorar hábitos saludables.

## Características

* Registro de alimentos y sus calorías
* Registro de ejercicios y calorías quemadas
* Cálculo automático del balance calórico
* Interfaz sencilla y fácil de usar

## Tecnologías utilizadas

* React
* TypeScript
* Vite
* CSS / Tailwind

## Instalación

1. Clona el repositorio:

```bash
git clone https://github.com/danielaRen21/Colories-Tacker.git
```

2. Instala las dependencias:

```bash
npm install
```

3. Ejecuta el proyecto en desarrollo:

```bash
npm run dev
```

## Uso

* Agrega los alimentos consumidos indicando sus calorías.
* Registra los ejercicios realizados.
* Visualiza el balance diario entre calorías consumidas y quemadas.

## Estado del proyecto

Proyecto en desarrollo con fines educativos y de práctica.
